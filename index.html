austimes PyPI


Private Python package index for the austimes organisation.
Install packages with: 

pip install <package_name> --extra-index-url https://austimes.github.io/pypi/
or with uv: 
uv add <package_name> --extra-index-url https://austimes.github.io/pypi/

Packages are hosted in private GitHub repositories. Ensure you have run gh auth login and gh auth setup-git for authentication.
